(Pre-)Publications

[2] The constant scalar curvature Kähler metric on bounded pseudoconvex domains, preprint, latest version, 2025

We post the following question which might be interesting to understand the cscK metric on complete manifolds, please see the note for a detailed explaination:
Let \((M,\omega_{KE})\) be a complete Kähler manifold endowed with a Kähler-Einstein metric \(\omega_{KE}\) of \(C^0\)-bounded geometry. Suppose there is a \(\varphi \in C^2(M)\) such that \(\omega_{\mathrm{cscK}} = \omega_{KE} + \sqrt{-1}\partial\bar{\partial}\varphi\) defines a complete cscK metric, and the volume forms satisfy \(\omega_{\mathrm{cscK}}^n = e^F \omega_{KE}^n\) for some function \(F\in C^0(M)\). If for any point $p \in M$, we have \[ \lim_{r \to \infty} \|F\|_{C^0(M \setminus B(p,r))} = 0 . \] Then, \(\omega_{\mathrm{cscK}} = \omega_{KE}\).
It is not difficult to see that if $\omega_{KE}$ is Calabi-Yau, then this question is true. So, only the case of negative scalar curvature needs to be considered.
